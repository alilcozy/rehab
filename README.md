# rehab

The Mini HUb - A collection of small projects built to practise and rebuild confidence in HTML, CSS, and JavaScript.

> Note: I was logged into a school account on VS Code during development, so commits on this account are non exsistent. (sorry!). Please read the dev log on flavourtown if you want to read about my journey.

## Projects

### Clock
A simple real-time clock. Built primarily to shake off rust and get comfortable with JavaScript's date and time methods again.

### Calculator
A fully functional calculator built using stacks data structure to handle operator precedence correctly. My first time using Grid properly.

### Coin Flip
A coin flip app with CSS flip animations. Helped me build more confidence working with CSS transitions and keyframe animations.

### Weather App
A weather app that fetches live data from the Open-Meteo API.
